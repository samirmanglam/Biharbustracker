INSTALL KAISE KAREIN

Pehli baar karne ka step-by-step guide

1
APK Download Karo

Upar "Driver APK Download" button dabao. File tumhare phone mein save ho jayegi.

2
Unknown Sources Allow Karo

Settings → Security → "Install Unknown Apps" → Chrome ya File Manager → Allow karo.

3
APK Install Karo

Downloads folder mein jao, "driver-app.apk" file pe tap karo → Install dabao.

4
App Kholo & GPS Allow Karo

App pehli baar khulne par Location permission maangega — "Allow all the time" chuno.

5
Route Select Karo aur Start Karo!

Apna route chuno → "Trip Shuru Karo" dabao → Bus live ho gayi! 🎉
